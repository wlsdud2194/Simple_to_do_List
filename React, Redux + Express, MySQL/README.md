# Todo 노트

지금 까지 배운 것을 이용하여 Todo 리스트와 RESTful API를 만들어 간단한 웹 프로토콜을 만들었습니다.


### Frontend

- React.js

- Redux ( react-redux, immutable, redux-actions, redux-thunk  )

- axios

- Sass



### Backend

- Node.js

- Express.js

- MariaDB
